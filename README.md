## Как собрать и запустить rasa

- Сделать окружение.
- Установить зависимости rasa, transformers.
- Войти в окружение.
- Провести тренировку ```rasa train```.
- Запустить бота командой ```rasa run --enable-api --cors "*" -vv --log-file log_file.txt```
